## 🚀 30초 즉시 해결법 (가장 빠름!)

### WordPress Elementor 편집 페이지에서:

1. **F12 키 누르기** (개발자 도구 열기)
2. **Console 탭 클릭**
3. **아래 코드 복사 → 붙여넣기 → Enter**

```javascript
// 모든 Elementor 쿠키 삭제
document.cookie.split(";").forEach(c => {
    const name = c.split("=")[0].trim();
    if(name.includes("elementor")) {
        document.cookie = name + "=;expires=Thu, 01 Jan 1970 00:00:00 UTC;path=/;";
    }
});

// 로컬 스토리지 삭제
Object.keys(localStorage).forEach(k => k.includes("elementor") && localStorage.removeItem(k));

// 새로고침
alert("✅ 완료! 3초 후 새로고침됩니다.");
setTimeout(() => location.reload(), 3000);
```

4. **3초 대기 → 자동 새로고침**
5. **완료!** ✅

## 📋 여전히 안 될 때: 영구 해결

### Code Snippets로 영구 방지

1. **WordPress 관리자** 접속
2. **플러그인 → 새로 추가**
3. **"Code Snippets" 검색 → 설치 → 활성화**
4. **Snippets → Add New**
5. 제목: `안전 모드 제거`
6. 아래 코드 붙여넣기:

```php
<?php
add_action('init', function() {
    if (isset($_COOKIE['elementor_safe_mode'])) {
        setcookie('elementor_safe_mode', '', time() - 3600, '/');
        unset($_COOKIE['elementor_safe_mode']);
    }
}, 1);

add_filter('elementor/editor/localize_settings', function($settings) {
    $settings['initial_document']['safe_mode'] = false;
    return $settings;
}, 999);

add_action('admin_init', function() {
    delete_user_meta(get_current_user_id(), '_elementor_safe_mode');
    delete_transient('elementor_safe_mode_' . get_current_user_id());
});
?>
```

7. **"Save Changes and Activate" 클릭**
8. **브라우저 캐시 삭제** (Ctrl+Shift+Delete)
9. **WordPress 다시 로그인**
10. **완료!** ✅

## 🔍 그래도 안 될 때

### 브라우저 완전 클린

```
Chrome:
1. Ctrl + Shift + Delete

3. 모든 항목 체크
4. "데이터 삭제"

또는:
- Incognito 모드로 WordPress 접속
- 다른 브라우저(Firefox, Edge) 시도
```

### 플러그인 충돌 확인

```
1. 플러그인 → 설치된 플러그인
2. Elementor 외 모든 플러그인 비활성화
3. Elementor 편집기 열기
4. 정상 작동하면 플러그인을 하나씩 활성화하며 범인 찾기
```
