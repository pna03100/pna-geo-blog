## 🎯 추가로 할 것 (선택사항)

### Code Snippets로 CORS 헤더 추가

위에서 wp-config.php를 수정했다면, 이제 CORS 헤더만 추가하면 완벽!

1. **WordPress 관리자 → 플러그인 → 새로 추가**
2. **"Code Snippets" 검색 → 설치 → 활성화**
3. **Snippets → Add New**
4. **코드 붙여넣기:**

```php
<?php
add_action('init', function() {
    $allowed = array(
        'https://cms.pnamarketing.co.kr',
        'https://pnamarketing.co.kr'
    );
    
    $origin = $_SERVER['HTTP_ORIGIN'] ?? '';
    
    if (in_array($origin, $allowed)) {
        header("Access-Control-Allow-Origin: $origin");
        header("Access-Control-Allow-Credentials: true");
    }
}, 1);

add_action('elementor/editor/before_enqueue_scripts', function() {
    echo '<script>
    window.addEventListener("error", function(e) {
        if (e.message && e.message.includes("cross-origin")) {
            e.preventDefault();
            return true;
        }
    }, true);
    </script>';
}, 1);
?>
```

5. **"Save Changes and Activate" 클릭**
