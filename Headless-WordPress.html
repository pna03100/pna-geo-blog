✅최종 체크리스트

( 출시 직전 / 작업 완료 후 확인용)

⬜ 1. 실시간 색인 (Indexing API)

  • 글 발행과 동시에 Google Indexing API가 호출되는가?
  • 발행 후 1분 이내 구글 검색에서 URL이 노출되는가?
  • GSC 색인 상태가 “발견됨/색인됨”으로 정상 표시되는가?

⬜ 2. AI 인용 구조 (Answer Block & TL;DR)

  • 본문 최상단에 TL;DR 요약 블록이 존재하는가?
  • 각 주요 문단의 첫 문장이 질문에 바로 답하는 문장인가?
  • TL;DR과 Answer Block만 읽어도 글의 핵심이 전달되는가?

⬜ 3. 엔티티 연결 (SameAs 스키마)

  • Organization / Person 스키마가 정상 출력되는가?
  • sameAs에 공식 SNS / 프로필 링크가 연결되어 있는가?
  • 리치 결과 테스트에서 스키마 오류가 0건인가?

⬜ 4. 성능 극대화 (속도 & 이미지)

  • 모바일 PageSpeed 점수가 95점 이상인가?
  • 주요 이미지가 AVIF 포맷으로 제공되는가?
  • LCP 요소가 불필요한 이미지/모션에 의해 지연되지 않는가?

⬜ 5. 데이터 통로 무결성 (Data-First)

  • 디자인 작업 전에 WordPress 데이터가 실제 화면에 출력되는 것을 확인했는가?
  • 데이터 깨짐/누락 없이 제목·본문·이미지가 정상 표시되는가?
  • 임시 더미 데이터로 디자인하지 않았는가?
