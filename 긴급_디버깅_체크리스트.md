# 🚨 긴급 디버깅 체크리스트

## 📋 정확히 무엇이 안 되나요?

아래 중 해당하는 항목에 체크해주세요:

### A. Elementor 도구 메뉴를 못 찾겠어요
- [ ] WordPress 왼쪽 메뉴에 "Elementor" 메뉴가 아예 없음
- [ ] "Elementor" 메뉴는 있는데 "도구"가 없음
- [ ] 직접 URL로도 접속이 안 됨

### B. WordPress 설정을 변경했는데 여전히 에러
- [ ] 도메인 설정을 cms로 바꿨는데 여전히 CORS 에러
- [ ] Code Snippets 코드를 추가했는데도 에러
- [ ] 브라우저 캐시도 삭제했는데 여전히 에러

### C. 새로운 에러가 발생함
- [ ] 설정 변경 후 사이트가 아예 안 열림
- [ ] 다른 에러 메시지가 나타남
- [ ] WordPress 관리자 접속이 안 됨

### D. Elementor 편집기가 여전히 이상함
- [ ] 편집기가 열리는데 위젯이 안 보임
- [ ] 드래그 앤 드롭이 안 됨
- [ ] 저장이 안 됨

## ⚡ 단계별 재확인

아래를 **순서대로** 체크하며 진행해주세요:

### ✅ 단계 1: WordPress 접속 확인
```
https://cms.pnamarketing.co.kr/wp-admin
```
- [ ] 접속 가능
- [ ] 접속 불가능 (에러 메시지: _______)

### ✅ 단계 2: Elementor 플러그인 확인
```
플러그인 → 설치된 플러그인
```
- [ ] "Elementor" 플러그인이 "활성화" 되어 있음
- [ ] "Elementor" 플러그인이 "비활성화" 되어 있음
- [ ] "Elementor" 플러그인이 아예 없음

### ✅ 단계 3: 도메인 설정 확인
```
설정 → 일반
```
현재 값 (스크린샷 또는 텍스트로):
- WordPress 주소(URL): __________
- 사이트 주소(URL): __________
